Don't Repeat Ourselves

scanner

The ultimate goal of webshell is to modularize scattered features reimplemented in so many WebView-based components and reuse these at will.

Leveraging WebViews

important_devices

Implement, test and compose rich behaviors embedded in injected scripts to craft amazing, reliable WebView-based components.

Autoheight Done Right

height

Discorver a landmark use-case: the Autoheight WebView. We depict caveats and feature workarounds to convey transparency over magic and let API consumers decide.

Written in Typescript

We are using advanced typing techniques to augment components with props in HOCs. 80% of this package codebase are types, resulting in a small bundle and powerful intellisense.

Declarative API

integration_instructions

Create instances of features, pass them to makeWebshell HOC, and enjoy new props and behaviors! The API is terse, and yet can be extended at will.

React Native 0.59+

This library provides hooks and thus requires React Native 0.59 or greater, and expo SDK 33.0 or greater.
